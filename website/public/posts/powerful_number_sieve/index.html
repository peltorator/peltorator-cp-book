–ü—É—Å—Ç—å $g(n)$ –∏ $h(n)$ ‚Äî —Ç–∞–∫–∏–µ –º—É–ª—å—Ç–∏–ø–ª–∏–∫–∞—Ç–∏–≤–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏, —á—Ç–æ $f = h*g$. –¢–æ–≥–¥–∞ –≤–µ—Ä–Ω–∞ —Å–ª–µ–¥—É—é—â–∞—è –ª–µ–º–º–∞:

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ: –í —Å—É–º–º–µ —Å–ª–µ–≤–∞ –µ—Å—Ç—å –≤—Å–µ –≤–æ–∑–º–æ–∂–Ω—ã–µ —Å–ª–∞–≥–∞–µ–º—ã–µ –≤–∏–¥–∞ $h(k) \cdot g(l)$, –≥–¥–µ $k \cdot l \le n$. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –¥–ª—è —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ $k$ –º—ã –¥–æ–ª–∂–Ω—ã –ø–µ—Ä–µ–º–Ω–æ–∂–∏—Ç—å $h(k)$ —Å–æ –≤—Å–µ–º–∏ —Ç–∞–∫–∏–º–∏ $g(l)$, —á—Ç–æ $l \le \left\lfloor \frac{n}{k} \right\rfloor$, —Ç–æ –µ—Å—Ç—å —Å –ø—Ä–µ—Ñ–∏–∫—Å–Ω–æ–π —Å—É–º–º–æ–π $g$. –ß—Ç–æ –∏ —Ç—Ä–µ–±–æ–≤–∞–ª–æ—Å—å –¥–æ–∫–∞–∑–∞—Ç—å.

–ü—É—Å–∫–∞–π —Ñ—É–Ω–∫—Ü–∏—è $h$ —Ç–∞–∫–æ–≤–∞, —á—Ç–æ $h(p) = 0$ –¥–ª—è –ª—é–±–æ–≥–æ –ø—Ä–æ—Å—Ç–æ–≥–æ $p$. –¢–æ–≥–¥–∞ –∑–∞–º–µ—Ç–∏–º, —á—Ç–æ –≤ –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ –≤—ã—Ä–∞–∂–µ–Ω–∏—è –∏–∑ –ª–µ–º–º—ã —Ç–µ —Å–ª–∞–≥–∞–µ–º—ã–µ, –≤ –∫–æ—Ç–æ—Ä—ã—Ö $k$ —Å–æ–¥–µ—Ä–∂–∏—Ç –≤ —Å–µ–±–µ –Ω–µ–∫–æ—Ç–æ—Ä—ã–π –ø—Ä–æ—Å—Ç–æ–π –º–Ω–æ–∂–∏—Ç–µ–ª—å —Ä–æ–≤–Ω–æ –≤ –ø–µ—Ä–≤–æ–π —Å—Ç–µ–ø–µ–Ω–∏, –∑–∞–Ω—É–ª—è—é—Ç—Å—è (–∏–∑ —Å–≤–æ–π—Å—Ç–≤–∞ —Ñ—É–Ω–∫—Ü–∏–∏ $h$). –û—Å—Ç–∞–ª—å–Ω—ã–µ –∂–µ —á–∏—Å–ª–∞ —Å–æ–¥–µ—Ä–∂–∞—Ç –∫–∞–∂–¥—ã–π —Å–≤–æ–π –ø—Ä–æ—Å—Ç–æ–π –º–Ω–æ–∂–∏—Ç–µ–ª—å —Ö–æ—Ç—è –±—ã –≤–æ –≤—Ç–æ—Ä–æ–π —Å—Ç–µ–ø–µ–Ω–∏. –ë—É–¥–µ–º –Ω–∞–∑—ã–≤–∞—Ç—å —Ç–∞–∫–∏–µ —á–∏—Å–ª–∞ —Å–∏–ª—å–Ω—ã–º–∏ (–æ—Ç –∞–Ω–≥–ª–∏–π—Å–∫–æ–≥–æ powerful, –∫–æ—Ç–æ—Ä–æ–µ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –æ—Ç —Å–ª–æ–≤–∞ power –≤ –∑–Ω–∞—á–µ–Ω–∏–∏ «—Å—Ç–µ–ø–µ–Ω—å»).

–õ–µ–º–º–∞: –í—Å—è–∫–æ–µ —Å–∏–ª—å–Ω–æ–µ —á–∏—Å–ª–æ $n$ –º–æ–∂–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç—å –≤ –≤–∏–¥–µ $a^2b^3$ –¥–ª—è –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö –Ω–∞—Ç—É—Ä–∞–ª—å–Ω—ã—Ö $a$ –∏ $b$.

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ: –î–ª—è —ç—Ç–æ–≥–æ –Ω—É–∂–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç—å –∫–∞–∂–¥—É—é —Å—Ç–µ–ø–µ–Ω—å –ø—Ä–æ—Å—Ç–æ–≥–æ $p_i^{\alpha_i}$, –≤—Ö–æ–¥—è—â—É—é –≤ $n$, –≤ —Ç–∞–∫–æ–º –≤–∏–¥–µ, –∞ –ø–æ—Ç–æ–º –ø–µ—Ä–µ–º–Ω–æ–∂–∏—Ç—å –∏—Ö. –¢–æ –µ—Å—Ç—å –Ω–∞–¥–æ –ø—Ä–æ—Å—Ç–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç—å —Å—Ç–µ–ø–µ–Ω—å $\alpha_i$ –≤ –≤–∏–¥–µ $2x + 3y$. –ë–µ–∑—É—Å–ª–æ–≤–Ω–æ, —ç—Ç–æ –º–æ–∂–Ω–æ —Å–¥–µ–ª–∞—Ç—å, —Ç–∞–∫ –∫–∞–∫ $\alpha_i \ge 2$ (–¥–ª—è —á–µ—Ç–Ω–æ–≥–æ $n$ –º–æ–∂–Ω–æ –≤–∑—è—Ç—å $x = \frac{n}{2}$ –∏ $y = 0$, –∞ –¥–ª—è –Ω–µ—á–µ—Ç–Ω–æ–≥–æ –ø–æ–¥–æ–π–¥–µ—Ç —Ä–µ—à–µ–Ω–∏–µ $x = \frac{n - 3}{2}$ –∏ $y = 1$).

–¢–æ–≥–¥–∞ –¥–ª—è –≤—ã—á–∏—Å–ª–µ–Ω–∏—è $s_f$ –Ω–∞–º –Ω—É–∂–Ω–æ –Ω–∞—É—á–∏—Ç—å—Å—è —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ –ø–µ—Ä–µ–±–∏—Ä–∞—Ç—å –≤—Å–µ —Å–∏–ª—å–Ω—ã–µ —á–∏—Å–ª–∞ $k$ –∏ —Å—É–º–º–∏—Ä–æ–≤–∞—Ç—å $h(k) s_g\left(\left\lfloor \frac{n}{k} \right\rfloor\right)$. –ë—É–¥–µ–º –¥–µ–ª–∞—Ç—å —ç—Ç–æ —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ, –ø–µ—Ä–µ–±–∏—Ä–∞—è –ø—Ä–æ—Å—Ç—ã–µ —á–∏—Å–ª–∞ –ø–æ –≤–æ–∑—Ä–∞—Å—Ç–∞–Ω–∏—é –∏ –≤—ã–±–∏—Ä–∞—è –∏—Ö –≤ —Å—Ç–µ–ø–µ–Ω–∏ –ª–∏–±–æ –Ω–æ–ª—å, –ª–∏–±–æ —Ö–æ—Ç—è –±—ã –¥–≤–∞. –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ –Ω–∞ —Ç–æ, —á—Ç–æ –µ—Å–ª–∏ –ø—Ä–æ—Å—Ç–æ–µ —á–∏—Å–ª–æ $p$ –≤—Ö–æ–¥–∏—Ç –≤ —Ä–∞–∑–ª–æ–∂–µ–Ω–∏–µ —Å–∏–ª—å–Ω–æ–≥–æ —á–∏—Å–ª–∞ $k$, —Ç–æ $p \le \sqrt{k} \le \sqrt{n}$, –ø–æ—Ç–æ–º—É —á—Ç–æ –∫–∞–∂–¥—ã–π –ø—Ä–æ—Å—Ç–æ–π –º–Ω–æ–∂–∏—Ç–µ–ª—å –≤—Ö–æ–¥–∏—Ç –≤ $k$ —Ö–æ—Ç—è –±—ã –≤–æ –≤—Ç–æ—Ä–æ–π —Å—Ç–µ–ø–µ–Ω–∏, –ø–æ—ç—Ç–æ–º—É –ø–µ—Ä–µ–±–∏—Ä–∞—Ç—å –ø—Ä–æ—Å—Ç—ã–µ –º—ã –º–æ–∂–µ–º —Ç–æ–ª—å–∫–æ –¥–æ $\sqrt{n}$.

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º —Å–ª–µ–¥—É—é—â–∏–π –∞–ª–≥–æ—Ä–∏—Ç–º –≤—ã—á–∏—Å–ª–µ–Ω–∏—è —Ñ—É–Ω–∫—Ü–∏–∏ $s_f$:

T s_f(long long n, long long cur, int idx, bool new_value = true) {
    T ans = new_value ? h(cur) * s_g(n / cur) : 0;
    if (n / cur / primes[idx] / primes[idx] == 0) { // can't multiply anymore
        return ans;
    }
    ans += s_f(n, cur, idx + 1, false);
    cur = cur * primes[idx] * primes[idx];
    while (cur <= n) {
        ans += s_f(n, cur, idx + 1, true);
        cur *= primes[idx];
    }
    return ans;
}

–ó–¥–µ—Å—å primes ‚Äî –º–∞—Å—Å–∏–≤ —Å –ø—Ä–æ—Å—Ç—ã–º–∏ —á–∏—Å–ª–∞–º–∏ —Ö–æ—Ç—è –±—ã –¥–æ $\sqrt{N}$; cur ‚Äî —Ç–µ–∫—É—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ $k$, –∫–æ—Ç–æ—Ä–æ–µ –º—ã –ø–µ—Ä–µ–±–∏—Ä–∞–µ–º —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ; idx ‚Äî –∏–Ω–¥–µ–∫—Å —Ç–µ–∫—É—â–µ–≥–æ –ø—Ä–æ—Å—Ç–æ–≥–æ –∏–∑ —Å–ø–∏—Å–∫–∞ primes; new_value ‚Äî —Ñ–ª–∞–≥, –∫–æ—Ç–æ—Ä—ã–π –Ω—É–∂–µ–Ω, —á—Ç–æ–±—ã –º—ã –Ω–µ –ø—Ä–∏–±–∞–≤–∏–ª–∏ –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∫ –æ—Ç–≤–µ—Ç—É –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ä–∞–∑, –∫–æ–≥–¥–∞ –º—ã –Ω–µ –±–µ—Ä–µ–º —Ç–µ–∫—É—â–µ–µ –ø—Ä–æ—Å—Ç–æ–µ –≤ —Ä–∞–∑–ª–æ–∂–µ–Ω–∏–µ; –∞ —Ç–∏–ø T –º–æ–∂–µ—Ç –±—ã—Ç—å –ª—é–±—ã–º —Ç–∏–ø–æ–º –Ω–∞ –í–∞—à –≤—ã–±–æ—Ä: –Ω–∞–ø—Ä–∏–º–µ—Ä, long long –∏–ª–∏ –í–∞—à –∫–ª–∞—Å—Å –¥–ª—è –º–æ–¥—É–ª—å–Ω–æ–π –∞—Ä–∏—Ñ–º–µ—Ç–∏–∫–∏.

–õ–µ–≥–∫–æ –≤–∏–¥–µ—Ç—å, —á—Ç–æ —ç—Ç–æ—Ç –∞–ª–≥–æ—Ä–∏—Ç–º –≤—ã—á–∏—Å–ª—è–µ—Ç $s_f$, –ø–µ—Ä–µ–±–∏—Ä–∞—è –ª–∏—à—å —Å–∏–ª—å–Ω—ã–µ —á–∏—Å–ª–∞. –û—Ü–µ–Ω–∏–º –≤—Ä–µ–º—è –µ–≥–æ —Ä–∞–±–æ—Ç—ã. –ë—É–¥–µ–º —Å—á–∏—Ç–∞—Ç—å, —á—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ $s_g$ –≤ —Ç–æ—á–∫–µ $n$ –º—ã –º–æ–∂–µ–º –≤—ã—á–∏—Å–ª–∏—Ç—å –∑–∞ –≤—Ä–µ–º—è $T_{s_g}(n)$, –∞ –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ $h(cur)$ —Ä–∞–±–æ—Ç–∞–µ—Ç –∑–∞ $O(1)$ (–æ–Ω–æ –º–æ–∂–µ—Ç —Å—á–∏—Ç–∞—Ç—å—Å—è –Ω–∞ –ª–µ—Ç—É –ø–æ –º—É–ª—å—Ç–∏–ø–ª–∏–∫–∞—Ç–∏–≤–Ω–æ—Å—Ç–∏ —á–µ—Ä–µ–∑ –∑–Ω–∞—á–µ–Ω–∏—è –≤ —Å—Ç–µ–ø–µ–Ω—è—Ö –ø—Ä–æ—Å—Ç—ã—Ö). –¢–æ–≥–¥–∞

$$T_{s_f(n)} = \sum_{k\text{ is powerful}}T_{s_g}\left(\left\lfloor\frac{N}{k}\right\rfloor\right)\leq\sum_{a, b : a^2b^3\leq N}T_{s_g}\left(\frac{N}{a^2b^3}\right)\approx\int\limits_{1}^{\sqrt{N}}\int\limits_{1}^{\sqrt[3]{N/a^2}}T_{s_g}\left(\frac{N}{a^2b^3}\right)\ \mathrm{d}b\ \mathrm{d}a$$

–î–ª—è –ø—Ä–æ—Å—Ç–æ—Ç—ã –ø—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º, —á—Ç–æ $T_{s_g}(n) = n^c$, –≥–¥–µ $c > 1/3$. –¢–æ–≥–¥–∞

$$ \int\limits_{1}^{\sqrt{N}}\int\limits_{1}^{\sqrt[3]{N/a^2}}T_{s_g}\left(\frac{N}{a^2b^3}\right)\ \mathrm{d}b\ \mathrm{d}a = \int\limits_{1}^{\sqrt{N}}\int\limits_{1}^{\sqrt[3]{N/a^2}}\frac{N^c}{a^{2c}b^{3c}}\ \mathrm{d}b\ \mathrm{d}a = N^c \int\limits_{1}^{\sqrt{N}} a^{-3c} \int\limits_{1}^{\sqrt[3]{N / a^2}} b^{-3c} \ \mathrm{d}b\ \mathrm{d}a = $$

$$ = N^c \int\limits_{1}^{\sqrt{N}} a^{-2c} \cdot (3c - 1) \cdot (1 - const_1) \ \mathrm{d}a \le $$

$$ \le (3c - 1)N^c\int\limits_{1}^{\sqrt{N}} a^{-2c} \ \mathrm{d}a\leq \begin{cases} (1 - 2c)(3c - 1)N^c\cdot N^{1/2 - c} = O(\sqrt{N}), & c < \frac{1}{2}, \\
(3c - 1)N^c\ln\sqrt{N} = O(\sqrt{N}\log{n}), & c = \frac{1}{2}, \\
(3c - 1)(2c - 1)N^c = O(T_{s_g}(N)), & c > \frac{1}{2}. \end{cases} $$

–ì–¥–µ $const_1$ ‚Äî –∫–∞–∫–æ–µ-—Ç–æ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–µ —á–∏—Å–ª–æ.

–û—á–µ–≤–∏–¥–Ω–æ, –µ—Å–ª–∏ $c \le 1/3$, —Ç–æ –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã –∏ –ø–æ–¥–∞–≤–Ω–æ —Å–æ—Å—Ç–∞–≤–ª—è–µ—Ç $O(\sqrt{N})$.

–ü—Ä–∏–º–µ—Ä: –ï—Å–ª–∏ $f(n)=\mathrm{rad}(n)$, —Ç–æ –µ—Å—Ç—å $f(p_1^{\alpha_1} \cdot \ldots \cdot p_k^{\alpha_k}) = p_1p_2 \cdot \ldots \cdot p_k$, —Ç–æ –≤ –∫–∞—á–µ—Å—Ç–≤–µ $h(n)$ –º–æ–∂–Ω–æ –≤–∑—è—Ç—å –º—É–ª—å—Ç–∏–ø–ª–∏–∫–∞—Ç–∏–≤–Ω—É—é —Ñ—É–Ω–∫—Ü–∏—é, –¥–ª—è –∫–æ—Ç–æ—Ä–æ–π –≤–µ—Ä–Ω–æ $h(1) = 1$, $h(p) = 0$ –∏ $h(p^k) = p - p^2$ –¥–ª—è –≤—Å–µ—Ö $k > 1$, –∞ –≤ –∫–∞—á–µ—Å—Ç–≤–µ $g$ –º–æ–∂–Ω–æ –≤–∑—è—Ç—å $g(n) = n$. –õ–µ–≥–∫–æ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å, —á—Ç–æ $f = g * h$. –¢–æ–≥–¥–∞ —Å—É–º–º–∞ $f(n)$ –Ω–∞ –ø—Ä–µ—Ñ–∏–∫—Å–µ —Ä–∞–∑–º–µ—Ä–∞ $N$ –±—É–¥–µ—Ç —Å—á–∏—Ç–∞—Ç—å—Å—è –∑–∞ $O(\sqrt{N})$, –ø–æ—Ç–æ–º—É —á—Ç–æ $T_{s_g}(n) = O(1)$.

–ó–∞–º–µ—á–∞–Ω–∏–µ: –ö–∞–∫ –≤–∏–¥–Ω–æ –∏–∑ –ø—Ä–∏–º–µ—Ä–∞, –ø—Ä–∏ –ø–æ–∏—Å–∫–µ —Ñ—É–Ω–∫—Ü–∏–π $h$ –∏ $g$ –∏–º–µ–µ—Ç —Å–º—ã—Å–ª –ø–æ–¥–æ–±—Ä–∞—Ç—å –∏—Ö –∑–Ω–∞—á–µ–Ω–∏—è –Ω–∞ —Å—Ç–µ–ø–µ–Ω—è—Ö –ø—Ä–æ—Å—Ç—ã—Ö.

–ó–∞–º–µ—á–∞–Ω–∏–µ: –ü—Ä–∏ –æ—Ü–µ–Ω–∫–µ –≤—Ä–µ–º–µ–Ω–∏ —Ä–∞–±–æ—Ç—ã –º—ã –ø–æ–ª—å–∑–æ–≤–∞–ª–∏—Å—å –ø—Ä–µ–¥—Å—Ç–∞–≤–∏–º–æ—Å—Ç—å—é —Å–∏–ª—å–Ω—ã—Ö —á–∏—Å–µ–ª –≤ –≤–∏–¥–µ $a^2b^3$. –ö–∞–∑–∞–ª–æ—Å—å –±—ã, –º–æ–∂–Ω–æ –±—ã–ª–æ –±—ã –Ω–∞–ø–∏—Å–∞—Ç—å –¥–≤–∞ for-–∞ –ø–æ —á–∏—Å–ª–∞–º $a$ –∏ $b$, –∏ —Ç–∞–∫–æ–π –∞–ª–≥–æ—Ä–∏—Ç–º –±—ã–ª –±—ã –ø—Ä–æ—â–µ —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ–≥–æ –ø–µ—Ä–µ–±–æ—Ä–∞ —Å–∏–ª—å–Ω—ã—Ö —á–∏—Å–µ–ª. –û–¥–Ω–∞–∫–æ —Å–∏–ª—å–Ω—ã–µ —á–∏—Å–ª–∞ –±—ã–≤–∞—é—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–∏–º—ã –≤ —Ç–∞–∫–æ–º –≤–∏–¥–µ –±–æ–ª—å—à–µ, —á–µ–º –æ–¥–Ω–∏–º —Å–ø–æ—Å–æ–±–æ–º, –ø–æ—ç—Ç–æ–º—É –ø—Ä–∏ —Ç–∞–∫–æ–º –∞–ª–≥–æ—Ä–∏—Ç–º–µ –ø—Ä–∏–¥—ë—Ç—Å—è –∫–∞–∫–∏–º-—Ç–æ –æ–±—Ä–∞–∑–æ–º –ø—Ä–æ–≤–µ—Ä—è—Ç—å —á–∏—Å–ª–∞ –Ω–∞ —É–Ω–∏–∫–∞–ª—å–Ω–æ—Å—Ç—å, —á—Ç–æ –º–æ–∂–µ—Ç –ø–æ–≤–ª–∏—è—Ç—å –Ω–∞ –≤—Ä–µ–º—è —Ä–∞–±–æ—Ç—ã.
